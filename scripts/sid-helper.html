Copy your ERPNext SID

Use this guide after the setup script opens ERPNext login.

Important: Stay on your ERPNext tab. Do not use localhost or redirect links. You only copy one cookie value, then return to the terminal.
  1. Log in to ERPNext in the other browser tab.
  2. Open DevTools with F12.
  3. Go to Application → Cookies → your ERP site.
  4. Click the row named sid.
  5. Copy the Value column (double-click, then Ctrl+C).
  6. Return to the terminal and press Enter — the setup script reads your clipboard automatically.
Edge / Chrome path: F12 → Application → Storage → Cookies → sid → Value
If clipboard auto-detect fails: paste the SID manually when the terminal asks for it.

Works with Cursor, Claude Desktop, Gemini CLI, Codex CLI, and OpenCode. SID sessions expire — run npm run setup-sid again when MCP stops working.
